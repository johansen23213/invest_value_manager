```

Cost of Debt (Kd):
- Average cost of debt = ~3.5% (A2A bond portfolio)
- Tax rate = 27% (Italian corporate)
- Kd after-tax = 3.5% x (1 - 0.27) = 2.6%

Capital Structure:
- Market Cap = EUR 7.9B

- Enterprise Value = EUR 13.2B
- Weight Equity (E/V) = 60%
- Weight Debt (D/V) = 40%

WACC = (60% x 8.2%) + (40% x 2.6%) = 4.9% + 1.0% = 5.9%

Sanity check: 5.9% WACC es razonable para utility EU regulada.
El WACC regulatorio de ARERA es ~5.5-6.0%, consistente.
Usare 6.5% para conservadurismo.
```

## 5. VALORACION MULTI-METODO (Framework v2.0)

**Tipo empresa: UTILITY (dividend income)**
**Metodos apropiados: DDM (60%) + EV/EBIT (40%)**

Nota: DCF basado en FCF no es apropiado porque FCF es negativo por ciclo de capex. Esto no es un problema estructural, pero hace el DCF inutilizable temporalmente.

### 5.1 METODO 1: Dividend Discount Model (DDM) - Peso 60%

A2A es dividend aristocrat (10+ anos sin corte, policy 4% growth)

**Gordon Growth Model:**
```
D0 = EUR 0.10 (dividendo actual)
g = 4% (policy minimo A2A)
D1 = D0 x (1 + g) = 0.10 x 1.04 = EUR 0.104

Required return (Ke) = 8.2% (derivado arriba)

Fair Value (DDM) = D1 / (Ke - g) = 0.104 / (0.082 - 0.04) = EUR 2.48
```

**Multi-stage DDM (mas conservador):**
```
Fase 1 (anos 1-5): g1 = 4% (policy)
Fase 2 (anos 6-10): g2 = 3% (fade)
Fase 3 (terminal): g3 = 2%

Valor aproximado: EUR 2.60-2.80
```

**DDM Fair Value: EUR 2.60 (promedio)**

### 5.2 METODO 2: EV/EBIT Normalizado - Peso 40%

**EBITDA 2025E:** EUR 2,400M (guidance raised en H1 2025)
**D&A:** ~EUR 900M
**EBIT 2025E:** EUR 1,500M

**Multiplo apropiado:**
| Comparable | EV/EBIT | Comentario |
|------------|---------|------------|
| Enel | ~10-11x | Mayor, mas diversificado |
| Hera | ~9-10x | Direct Italian peer |
| Iberdrola | ~12x | Premium por renewables |
| Sector median | ~10x | |

**Ajustes al multiplo:**
- Base sector: 10x
- Ajuste superior growth plan: +0.5x
- Ajuste municipal ownership (governance): -0.5x
- Ajuste Italia country: -0.5x
- **Multiplo final: 9.5x**

```
Fair EV = EBIT x Multiple = 1,500 x 9.5 = EUR 14,250M
Fair Equity = 14,250 - 5,300 (net debt) = EUR 8,950M
Fair Value per share = 8,950 / 3,150 = EUR 2.84
```

**EV/EBIT Fair Value: EUR 2.84**
