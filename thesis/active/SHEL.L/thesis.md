## Autocrítica v2.0

**Lo que el análisis original falló:**
1. Usó P/FCF sin ajustar por ciclo → sobre-estimó fair value
2. WACC asumido (8.7%) vs derivado (10.1%)
3. No usó EV/EBIT mid-cycle (método correcto para cíclicas)
4. Sizing 12-15% era excesivo para MoS real

**Aprendizaje:**
- Para cíclicas, EV/EBIT mid-cycle es más conservador y apropiado
- El P/FCF alto (14-15%) no significa "barata" si el múltiplo ya es bajo
- Beta >1 para commodities aumenta WACC significativamente
