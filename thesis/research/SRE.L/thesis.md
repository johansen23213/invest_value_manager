## 1. Resumen Ejecutivo

Sirius Real Estate es un REIT especializado en parques empresariales de valor anadido en Alemania (85%) y UK (15%), con 153 activos y 10,958 inquilinos. El modelo de negocio consiste en adquirir propiedades infrautilizadas, transformarlas mediante una plataforma interna (marketing digital + call center + ventas on-site), y crecer rentas + ocupacion.

**Por que esta barata:** Sector inmobiliario comercial europeo castigado por ciclo de tipos altos, preocupacion por refinanciacion de EUR 400M en junio 2026, y percepcion de activos "secundarios" vs prime offices.

**Mi contra-tesis:** La plataforma operativa interna genera ventaja competitiva sostenible, los activos secondary/business parks son menos afectados que offices prime por trabajo remoto, y el refinanciamiento ya esta cubierto con cash + RCF. El mercado sobre-descuenta el riesgo de tipos.

**Catalizador principal:** Refinanciacion exitosa de bond EUR 400M (junio 2026) + continuacion de LfL rent growth >5%.

**Fair Value (Expected):** 100p | Precio actual: 97.85 GBp | MoS: +2.2% (INSUFICIENTE)

## 2. Business Understanding

### 2.1 Modelo de Negocio

| Aspecto | Descripcion |
|---------|-------------|
| Problema que resuelve | Espacio de trabajo flexible y asequible para PYMES en Alemania/UK |
| Cliente target | SMEs, micro-empresas, startups que necesitan espacio industrial/oficina flexible |
| Como genera ingresos | Alquileres recurrentes (rentas mensuales/anuales) + servicios auxiliares |
| % ingresos recurrentes | >95% (rentas son el core) |

### 2.4 Por Que Esta Barata

**Narrativa del mercado:**
- [X] Sector out-of-favor (rotacion ciclica) - RE castigado por higher for longer
- [X] Riesgo refinanciamiento EUR 400M bond (junio 2026)
- [X] Activos "secundarios" percibidos como riesgo vs prime
- [X] Alemania en estancamiento economico (preocupacion por PYMES)
- [X] Dilution por equity raises (EUR 175M en 2024)

**Mi contra-tesis:**

| Mercado cree | Yo creo | Evidencia | P(equivocado) |
|--------------|---------|-----------|---------------|
| RE europeo seguira bajo presion | El sector secondary/business parks es resiliente vs offices | Ocupacion DE subiendo, LfL +5.2% | 30% |
| Refinanciamiento es riesgo | Ya esta cubierto: EUR 389M cash + EUR 150M RCF | Balance sheet solido, Fitch BBB | 15% |

| Alemania en recesion | Alemania proyecta +1.0-1.4% en 2026, PYMES son resilientes | IMF/Goldman forecasts | 30% |

### 2.5 Value Trap Checklist

| Factor | SI/NO | Comentario |
|--------|-------|------------|
| Industria en declive secular | NO | Business parks crecen, office decline no aplica |
| Disrupcion tecnologica inminente | NO | Trabajo remoto favorece flex space vs rigid offices |
| Management destruyendo valor | NO | 12 anos de dividendos crecientes, Fitch BBB |
| Balance deteriorandose (deuda subiendo, coverage bajando) | WATCH | Net LTV 38.3% (vs 31.4% prior), cost of debt subiendo |
| Insider selling masivo (>5% en 12m) | NO | Insiders comprando (Andrew Coombs 1,686 shares jul 2025) |
| Dividend cut reciente o probable | NO | +4% dividend increase en H1 2026 |
| Perdida market share >2pp en 3 anos | NO | Adquisiciones activas, market share creciendo |
| ROIC < WACC ultimos 3 anos | SI | ROC 4.6% vs WACC 10.3% (preocupante) |
| FCF negativo >2 anos consecutivos | NO | FFO positivo cada ano |
| Goodwill >50% equity | NO | Minimal goodwill |

**Total SI: 1 (ROIC issue)** - Pasa el checklist pero ROIC < WACC es senal de alarma que requiere investigacion.

**Nota sobre ROIC < WACC:** Este es un artifact de como se calcula ROIC para REITs. El ROC de 4.6% no captura el value-add model correctamente porque:
1. Las propiedades se compran a book value, no a market value
2. El WACC de 10.3% asume cost of equity alto que no refleja el bajo riesgo de alquileres estables
3. FFO yield del 7.4% vs cost of debt 2.5% = value creation real

### 4.3 WACC Derivado

```
Risk-free rate (10Y Bund): 2.3%
Beta: 0.75 (RE sector, less volatile)
ERP: 5.0%
Ke = 2.3% + 0.75 x 5.0% = 6.05%

Cost of debt (post-refi): 4.0% (vs 2.5% legacy)
Tax rate: ~20%
Kd after-tax = 4.0% x (1 - 0.20) = 3.2%

E/V = 60% | D/V = 40%

WACC = (60% x 6.05%) + (40% x 3.2%) = 4.9%
```

**Sanity check:** WACC 4.9% es bajo pero reasonable para un REIT defensive con BBB rating. Uso 6-7% para ser conservador.

## 5. Valoracion Multi-Metodo

### 5.1 Metodo 1: NAV (Peso 50%)

Para REITs, NAV es el metodo primario.

- EPRA NTA per share: EUR 1.1594 = 100.5p GBP
- Premium/discount tipico para business parks: 0.90x - 1.05x NAV
- Sirius tiene plataforma interna (merece premium) pero activos secondary (merece discount)
- **Fair multiple: 0.95x NAV**
- **Fair Value NAV: 95.5p**

### 5.2 Metodo 2: DDM (Peso 30%)

```
D0 = 6.15c (EUR) = 5.34p GBP
g = 3% (conservative, historico +6% pero dilution)
r = 8% (required return for Tier B REIT)

Fair Value = D1 / (r - g) = 5.34 x 1.03 / (0.08 - 0.03) = 5.50 / 0.05 = 110p
```

**Fair Value DDM: 110p** (mas optimista porque asume crecimiento sostenido)

### 5.3 Metodo 3: FFO Yield (Peso 20%)

- FFO/share FY25: 8.44c EUR = 7.32p GBP
- Target FFO yield para secondary RE: 7-8%
- **Fair Value at 7.5% FFO yield: 97.6p**
