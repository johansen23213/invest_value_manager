   - If management addresses AI agent impact on seat counts = forward-looking positive
   - SILENCE on this topic = potentially hiding early erosion data

   - Key metrics: TurboTax units filed, TurboTax Live attach rate, ARPU
   - IRS data shows 2026 season started strong (refunds up 10.9%, volumes tracking)

   - R3 identified 400bps drag on GBS (not 150bps as initially claimed)
   - If Mailchimp revenue declines >10% = KC#4 territory (recalculate FV excl goodwill)
   - If Mailchimp shows positive growth = material positive surprise (removes drag narrative)

6. **Operating margin expansion**
   - Guide is 28%+. Trend has been +2pp/yr (FY22: 20% -> FY25: 26%)
   - If >28% = operating leverage thesis confirmed
   - If <26% (regression) = AI spending eating margins without returns

### Tier 3: CONTEXT (informs background)

7. **Intuit Assist / AI adoption metrics**
   - Q1: 30% of QBO users adopted Intuit Assist (up from 10%)
   - If >40% = rapid adoption validates "AI as moat enhancer" thesis

8. **Credit Karma growth and monetization**
   - Underappreciated asset per moat assessment. FY25: +32%
   - Cross-sell with TurboTax/QBO strengthens ecosystem moat
   - If >+20% = platform flywheel working

9. **SBC as % of revenue and FCF**
   - R3 identified SBC at ~10% of revenue / 39% of FCF as material quality concern
   - If declining = quality improving. If rising = talent retention arms race
