CAPA 1 SCREENING — Detección de oportunidades 2 agentes · Ejecución paralela · Semanal + event-driven ▼
AGENT 1.1 · SCREENER CUANTITATIVO
Filtrado por métricas financieras y valoración
yfinance EDGAR SCORE 0-100
  • P/E · P/B · EV/EBITDA vs. sector
  • Net cash positivo (especial en biotechs)
  • Insider ownership > 10%
  • Market cap < 500M (small cap zone)
  • Momentum negativo reciente (contrarian)
  • Volumen mínimo para operativa viable
AGENT 1.2 · SCREENER SPECIAL SITUATIONS
Detección de eventos corporativos accionables
SEC RSS PR NEWSWIRE EVENT-DRIVEN
  • Merger/acquisition announced → spread arbitrage
  • Strategic review → posible liquidación
  • Bankruptcy filing → reorganización
  • Spin-off announced → stub valuation
  • Busted biotech / cash shell → net cash discount
  • Odd-lot tender offer → arbitraje pequeño
│
CAPA 2 ANÁLISIS — Due diligence en profundidad 5 agentes · Ejecución paralela · Por candidato seleccionado ▼
AGENT 2.1 · ANALISTA FINANCIERO
Valoración cuantitativa multi-método
DCFNAVNET CASH
  • DCF en 3 escenarios (base / bull / bear)
  • Net cash neto de deuda y liabilities
  • ROIC y márgenes evolución 5 años
  • Red flags contables (accruals, goodwill)
  • Margen de seguridad vs. precio actual
AGENT 2.2 · ANALISTA SPECIAL SITUATION
Modelado específico del evento corporativo
MERGER MODELLIQUIDATIONCVR
  • Merger: spread, prob. cierre, break fee, timeline
  • Liquidación: cash distribuible, haircuts activos
  • Spin-off: stub valuation, motivaciones mgmt
  • Biotech: burn rate, CVR modeling, reverse merger
  • Expected value + distribución de probabilidades
AGENT 2.3 · ANALISTA DE NEGOCIO
Análisis cualitativo y ventajas competitivas
MOATMANAGEMENTCATALYSTS
  • Modelo de negocio y posición competitiva
  • Management: track record + skin in the game
  • Catalizadores (corto y largo plazo)
  • Bull case / Bear case documentados
  • Riesgo disruption / regulatorio / concentración
AGENT 2.4 · ANALISTA DE RIESGO
Cuantificación de riesgos y sizing
RISK SCORESTOP LOSSMAX DD
  • Liquidez y bid-ask para sizing viable
  • ¿Qué tiene que ir mal para perder >30%?
  • Riesgo ejecución (timing, contraparte, regulatorio)
  • Exposición divisa (USD/GBP vs. EUR)
  • Max drawdown en escenario adverso
AGENT 2.5 · INVESTIGADOR WEB
Contexto reciente y fuentes externas
WEB SEARCH13FTRANSCRIPTS
  • Últimas noticias empresa y competidores
  • Earnings transcripts recientes
  • 13F filings: ¿otros value managers están?
  • Alpha Vulture / Seeking Alpha / blogs
  • Litigios, investigaciones regulatorias activas
│
CAPA 3 PORTFOLIO — Decisión y monitorización 2 agentes · Secuencial · Integra todos los outputs ▼
AGENT 3.1 · DECISION MAKER
Emisión de recomendación estructurada de inversión
BUY/SELL/PASSSIZINGCONVICTION
  • Investment Score compuesto 0–100 (agrega Capa 2)
  • Verifica: margen seguridad >30%, risk score <7
  • Correlación con posiciones actuales de cartera
  • Position size recomendado (% cartera)
  • Target price + stop loss + catalyst trigger
AGENT 3.2 · PORTFOLIO MONITOR
Vigilancia continua de posiciones abiertas
DAILYALERTSP&L TRACKING
  • Precio vs. tesis: ¿se está cumpliendo?
  • Alerta si precio cae >X% sin razón fundamental
  • Progreso de eventos (merger timeline, liquidación)
  • P&L real por posición (€, %, anualizado)
  • Detecta cambios en fundamentales (nuevos filings)

FLUJOS DE EJECUCIÓN

  4. Top 5 candidatos → análisis profundo
  5. Decision Maker emite recomendaciones

FLUJO 2 — ANÁLISIS BAJO DEMANDA
  1. Joan introduce ticker manualmente

  4. Decision Maker integra y puntúa
  5. Resultado guardado en Knowledge Base

FLUJO 3 — MONITORIZACIÓN DIARIA
  1. Cron trigger (día hábil, 18:00)
  2. Portfolio Monitor revisa posiciones abiertas
  3. Detecta alertas precio / noticias / filings
  4. Alerta crítica → notificación inmediata
  5. Cambio de tesis → propone revisión
  6. Actualiza P&L y métricas cartera
