# Decisión: RECHAZAR Sanofi (SAN.PA)

> Fecha: 2026-01-26
> Decisión: **NO COMPRAR - Pasar a Watchlist**
> Precio evaluado: €79.31

## Resumen Ejecutivo

Investment Committee RECHAZA compra de Sanofi en este momento. Margen de seguridad insuficiente (20-25% vs requerido >25%) para el nivel de riesgo (execution risk pipeline + patent cliff 2031-33 cierto). Estrategia: Watchlist, esperar validación pipeline con 25 readouts próximos 18 meses.

## Razones del Rechazo

### 1. Margen de Seguridad Insuficiente

**Sanofi**: 20-25% (€79 → €95-105 valor intrínseco)

**Comparación portfolio:**
- Shell: 30-35% ✅
- DTE: 30-35% ✅

Para un riesgo MAYOR, necesitamos margen MAYOR, no menor.

### 2. Riesgo Execution Demasiado Alto

**Patent cliff Dupixent 2031-33**: €16B (30% revenue) CIERTO que expira

**Pipeline 12 blockbusters**:
- Promesa: €10B incremental by 2030
- Histórico pharma Phase 3: 50% fracaso
- Probabilidad entregar 6-8 de 12: ~40-50%

**Si pipeline falla**: Valuación colapsa post-2031

### 3. Moat Temporal vs Durable

| Empresa | Moat | Durabilidad |
|---------|------|-------------|
| Shell | LNG infrastructure, escala global | 15+ años ✅ |
| DTE | T-Mobile stake + telecom infrastructure | 20+ años ✅ |
| Sanofi | Dupixent hasta 2031, luego pipeline | 5 años, luego ? ⚠️ |

### 4. Valuación Refleja Riesgo Real

P/E 10x (32% discount vs peers 15x) NO es "mercado equivocado".

Es mercado diciendo: **"Muéstranos que pipeline funciona PRIMERO"**

### 5. Hay Mejores Oportunidades

**Utilities EU (Enel, Iberdrola)**:
- Riesgo MENOR (regulado, estable)
- Dividendos 5% y 3.5% sostenibles
- Moat infrastructure durable
- Valuaciones razonables
- Re-rating cuando tipos bajen

**Por qué Utilities > Sanofi ahora:**
- Menor riesgo
- Moat más durable
- Yield comparable (Enel 5% vs Sanofi 4.85%)
- Portfolio más diversificado (energy + telecom + utility vs energy + telecom + pharma especulativo)

## Decisión Final

### ⏳ **PASAR A WATCHLIST**

**Comprar SI/CUANDO:**

1. **Amlitelimab aprueba FDA** (H1 2026) → valida pipeline capacity
2. **3+ readouts positivos** de los 25 próximos → aumenta probabilidad éxito total
3. **Precio cae a €70-75** → margen seguridad >30% compensa riesgo

**Configurar alertas:**
- Precio €75 → revisar para compra
- Amlitelimab FDA approval → evaluar compra
- Major pipeline readout positivo → evaluar

**Monitoring:**
- Q4 2025 earnings (enero 2026)
- 25 pipeline readouts 2025-26
- Dupixent quarterly growth
- Operating margin expansion

## Lección para el Sistema

**Aprendizaje:**
Margen de seguridad NO es negociable. Para riesgo execution alto (pharma pipeline, turnarounds, etc.), requerir margen >30%, NO aceptar 20-25%.

**Regla aplicar:**
```
IF execution_risk == "ALTO" AND margen_seguridad < 30%:
  → Watchlist, NO comprar
  → Esperar validación O mejor precio
```
