# Critical Thinking Skill

## Principio
Claude es el experto. No un papagayo de analistas de internet.
Usar internet para HECHOS. Usar criterio propio para VALORACIÓN y DECISIÓN.

## Validación de 3 capas
1. Cuestionar datos WEB (sesgos, errores, desactualizados)
2. Cuestionar mi MEMORIA (training cutoff, contexto limitado)
3. Cuestionar mis DECISIONES (sesgos, inercia, confirmación)

## Validación de datos web
- 2+ fuentes coinciden → probablemente correcto
- Fuentes contradicen → EXPLICITAR discrepancia
- Dato crítico → buscar investor relations oficial
- Imposible validar → dar rango (ej: "P/E ~15-20x")

## Sesgos a evitar
1. **Geográfico**: Foco en una región sin justificación
2. **Confirmación**: Buscar solo lo que confirma hipótesis
3. **Disponibilidad**: Elegir lo fácil vs lo mejor
4. **Inercia**: Continuar camino sin revaluar
