# Decision Template Sub-Skill

## Formato decisión investment committee

### Veredicto: BUY / WATCHLIST / REJECT

### Checklist
- [ ] Thesis completa y documentada
- [ ] Margen seguridad >25%: [valor]%
- [ ] Moat identificado: [tipo]
- [ ] Riesgos documentados: [count] riesgos
- [ ] Sizing dentro de límites: [valor]%
- [ ] Sector <25%: [sector] = [valor]%
- [ ] Geografía <35%: [geo] = [valor]%
- [ ] Cash post-compra >5%: [valor]%
- [ ] Autocrítica explícita: Sí/No

### Si BUY
Recomendación clara:
"Comprar €X de TICKER (Y% del portfolio).
Razón: [1-2 líneas]
Riesgo principal: [1 línea]
¿Confirmas para ejecutar en eToro?"

### Si WATCHLIST
- Precio objetivo de entrada: €X
- Condición de entrada: [qué debe pasar]
- Alerta configurada: Sí/No

### Si REJECT
- Razón principal: [1-2 líneas]
- ¿Revisitar en futuro? Sí/No
- Si sí, cuándo y por qué
